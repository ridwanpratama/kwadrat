Apa itu Kwadrat?
Kwadrat adalah aplikasi web untuk menghitung persamaan kuadrat,pertidaksamaan kuadrat dan pertidaksamaan linier.
Tentang Kwadrat
Fitur yang dimiliki kwadrat:
1. Menghitung Persamaan Kuadrat dengan Metode
- Metode 1 a(x - x1)(x - x2) = 0
- Metode 2 ax2 + bx + c = 0
- Metode 3 a(x - H)2 + K = 0
2. Jumlah dan hasil kali akar-akar persamaan kuadrat
- x1 + 2
- x1 * x2
- x12 + x22
- x12 + x22 / x1 * x2
